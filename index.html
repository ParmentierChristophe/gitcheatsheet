Git Cheat Shit

By crispy

Creation

Depuis un fichier existant

$ cd ~/projet/monprojet
$git init
$ git add .

Depuis un Repo existant

$ git clone ~/existing/repo/ ~/new/repo
$ git clone you@host.org:dir/projet.git
(le protocole par default est SSH)

changement locaux

Fichier changer dans le dossier en cours

$ git status

Les modifications apportées aux fichiers

$ git diff

Ajouter les changements en cours au prochain commit

$ git add .

Ajouter quelques modifications dans le fichier du prochain commit

$ git add -p nom_du_fichier

Valider toutes les modifications locales dans les fichiers

$ git commit -a

Commit le changement précédemment mis en scène

$ git commit

Historique commit

Voir tous les commits effectué depuis le debut

$ git log

montrer les changements au fil du temps pour un fichier spécifique

$ git blame nom_du_fichier

Branches & Tags

Liste toutes les branches

$ git branch -av

Changer de branche

$ git checkout nom_de_la_branche

Creer une nouvelle branche

$ git branch new_branch

Creer une branch basé sur votre remote

$ git checkout --track remote/nom_branch

Supprimer une branche

$ git branch -d nom_branch

Marquer le commit actuel avec un Tag

$ git tag nom_du_tag

Update et Publier

Liste toutes les remotes actuellement configurées

$ git remote -v

Voir les informations sur le remote

$ git remote show nom_du_remote

Supprimer une branche dans le remote

$ git branch -dr nom_remote/branche

Publier vos Tags

$ git push --tag

Merge & Rebase

Merge votre branche dans votre branche actuel

$ git merge nom_de_la_branche

Rebaser votre branche actuel dans la branche
Don't rebase published commits!

$ git rebase nom_de_la_branche

abandonner une rebase

$ git rebase --abort

Utilisez votre outil de merge configuré pour résoudre les conflits

$ git mergetool

Utilisez votre éditeur pour résoudre manuellement les conflits et (après avoir résolu) Fichier marque comme résolu

$ git add fichier_resolu $ git rm fichier_resolu

Annuler

Jeter tous les changements locaux dans votre répertoire de travail

$ git reset --hard HEAD

Jeter les changements locaux dans un fichier spécifique

$ git checkout HEAD nom_fichier

Revenir à un commit (en produisant un nouveau commit avec des changements contraires)

$ git revert nom_du_commit

Reset votre branche actuel à un commit precedent

$ git reset --hard nom_du_commit
